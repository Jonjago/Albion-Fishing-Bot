# Order Template Analysis Bot

This project is a **learning-focused software bot** developed to improve my skills in **Python** and **form / template recognition**.

The bot processes incoming orders that contain **article numbers**, analyzes them, and evaluates the data based on predefined **templates**.  
The main goal of this project is **education and experimentation**, not commercial use.

---

## 🎯 Purpose of the Project

This bot was created **solely for learning purposes**.

Through this project, I aimed to:
- Get more familiar with **Python**
- Learn how to work with **structured order data**
- Experiment with **template-based form recognition**
- Understand basic automation workflows for order processing

---

## ⚙️ Features

- Reads orders containing article numbers  
- Matches order data against predefined templates  
- Extracts and evaluates relevant information  
- Designed as a lightweight and modular learning project  

---

## 🖥️ Screen Resolution Requirement

⚠️ **Important:**  
To use this bot correctly, the **in-game screen resolution must be set to exactly `1920 x 1080`**, as shown in the demo video.

The bot relies on fixed screen positions and template recognition.  
Using a different resolution may cause incorrect detection or unexpected behavior.
